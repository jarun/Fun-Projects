# PrimePalindrome  
  
A fast C program to calculate the 1500th prime palindrome with 13 digits.  
  
# Execution time  
  
/* On Intel(R) Core(TM) i5-3210M CPU @ 2.50GHz */  
  
$ ./pp  
Thu Jul  2 00:28:26 IST 2015  
Thu Jul  2 00:32:17 IST 2015  
count: 1500 val: 1015834385101
